## 📝 STEP 1: CREATE DATABASE (2 MINUTES)

1. **Start XAMPP:**
   - Click "Start" for **Apache**
   - Click "Start" for **MySQL**

2. **Open phpMyAdmin:**
   ```
   http://localhost/phpmyadmin
   ```

3. **Create Database:**
   - Click "SQL" tab at the top
   - Open the file: `/database/setup_with_data.sql`
   - Copy **ALL** the code
   - Paste into the SQL box
   - Click "Go" button
   - ✅ Done! You'll see "church_management" database with 12 members and 12 payments!

---

## 🔧 STEP 2: CHECK CONFIG (30 SECONDS)

Open `/config/database.php` and verify:

```php
$host = 'localhost';
$db = 'church_management';
$user = 'root';
$pass = '';  // Empty for XAMPP
```

If it matches, you're good! ✅
