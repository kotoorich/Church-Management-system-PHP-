## 🐛 TROUBLESHOOTING

### Can't login?
- Make sure database is created
- Check MySQL is running in XAMPP
- Verify username: `admin`, password: `password`

### Blank page?
- Check Apache is running
- Files must be in `C:\xampp\htdocs\church_management\`
- Access via `http://localhost/church_management/`

### No data showing?
- Run the SQL file in phpMyAdmin
- Check database name is `church_management`
- Verify connection in `/config/database.php`

### Theme not changing?
- Clear browser cache (Ctrl+Shift+Del)
- Check browser console for JavaScript errors
- Try different browser
