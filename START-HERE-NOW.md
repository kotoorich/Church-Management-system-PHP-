## 📝 STEP 1: CREATE DATABASE (2 minutes)

1. Open XAMPP, start **Apache** and **MySQL**
2. Open browser: `http://localhost/phpmyadmin`
3. Click "SQL" tab
4. Open file: `/database/setup_with_data.sql`
5. Copy ALL the code
6. Paste into SQL tab
7. Click "Go"
8. ✅ Done! You'll see "church_management" database created!

---

## 🔧 STEP 2: CHECK DATABASE CONFIG (30 seconds)

Open `/config/database.php` - should look like this:

```php
$host = 'localhost';
$db = 'church_management';
$user = 'root';
$pass = '';  // Empty for XAMPP
```

If it matches, you're good! If not, change it to match above.

---

## 🎉 STEP 3: OPEN YOUR APP!

**Browser:** `http://localhost/church_management/`
